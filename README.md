# css_parser

This parser is based on the [CSS 2.1 specification](https://www.w3.org/TR/CSS2/)

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     css_parser:
       github: orklann/css_parser
   ```

2. Run `shards install`

## Usage

```crystal
require "css_parser"
```
