# cssparser

This parser is based on the [CSS Syntax Module Level 3](https://www.w3.org/TR/css-syntax-3/)

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     cssparser:
       github: orklann/cssparser
   ```

2. Run `shards install`

## Usage

```crystal
require "cssparser"
```

## Run tests

```
crystal spec
```
